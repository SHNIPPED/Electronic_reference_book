# Electronic Reference Book
Electronic Reference Book - Электоронный справочник организации
Полнофункциональное веб-приложение для управления справочником сотрудников с CRUD-операциями и аутентификацией.

## Технологический стек

*   **Frontend:** React, React Router
*   **Backend:** Node.js, Express
*   **Database:** MySQL
*   **Аутентификация:** JWT

## Возможности

*   Регистрация и вход пользователей (JWT).
*   Просмотр списка сотрудников.
*   Добавление, редактирование и удаление записей.

### Предварительные требования
*   Node.js (v18+)
*   MySQL (v8+)
*   Git
