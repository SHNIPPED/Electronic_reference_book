Electronic Reference Book - Электоронный справочник организации
Полнофункциональное веб-приложение для управления справочником сотрудников с CRUD-операциями и аутентификацией.

Технологический стек

Backend (API)
Node.js - среда выполнения JavaScript

Express.js (v4.21.2) 

MySQL + MySQL2 (v3.12.0) 

JSON Web Token (v9.0.2) 

CORS (v2.8.5) 

Body-parser (v1.20.3) 

Frontend (UI)
React (v18.3.1) 

React Router DOM (v7.1.2)

JSON Web Token (v9.0.2) 

База данных
MySQL 

Функциональность
CRUD-операции:
добавление новых сотрудников

 просмотр списка сотрудников и детальной информации

редактирование данных сотрудников

удаление сотрудников из системы

Безопасность:
JWT-токены для аутентификации

CORS политики для безопасности

Защита API endpoints
